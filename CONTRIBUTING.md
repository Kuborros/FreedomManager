
# How to contribute

Make a pull request. 

You should most likely build and create PRs based on current dev branch instead of the main one, unless they are identical or the fix should be pushed to master/main.

# AI Contributions Policy

Short answer: **NOT ALLOWED**

I do not accept contributions that had Artificial "Intelligence", specifically Large Language Models involved in _any_ step. 
This includes, but isn't limited to, GitHub Copilot, Zed, Cursor, Claude and other "Coding Assistants", as well as any code generation tools like GitHub Copilot or ChatGPT.

The generated code is not _yours_, it's an amalgamation of stolen content and i cannot accept it.

Here's a fun read for you: [Why not LLMS](https://codeberg.org/small-hack/open-slopware#why-not-llms)

Above includes every part of the project, not just commits and pull requests. If i smell a LLM use i reserve a right to close your PR on sight.
